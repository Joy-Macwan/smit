## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- MongoDB

### Build & Run

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd smit
   ```

2. **Quick build (all components)**
   ```bash
   ./build.sh
   ```

3. **Manual setup**

   **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

   **Backend:**
   ```bash
   cd backend
   npm install
   npm start
   ```
